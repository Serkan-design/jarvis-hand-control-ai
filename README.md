# 🤖 JARVIS Hand Control AI

AI-powered desktop assistant that lets you control your computer using **hand gestures** and **voice commands**.

---

## 🚀 Features

* ✋ Hand gesture mouse control
* 🖱️ Left click / Right click / Scroll / Alt+Tab
* 🎤 Voice commands (Turkish)
* 🔊 Text-to-Speech (JARVIS voice)
* 🖥️ Desktop automation (open apps, search, system control)
* 🎨 Futuristic HUD interface

---

## 🛠️ Technologies

* Python
* OpenCV
* MediaPipe
* PyAutoGUI
* SpeechRecognition
* Edge-TTS
* Pygame

---

## ⚡ Installation

### 1️⃣ Clone the repo

```bash
git clone https://github.com/Serkan-design/jarvis-hand-control-ai.git
cd jarvis-hand-control-ai
```

---

### 2️⃣ Install dependencies (TEK KOMUT)

```bash
pip install -r requirements.txt
```

---

### ❗ Eğer çalışmazsa (Windows için alternatif)

```bash
pip install opencv-python mediapipe pyautogui numpy edge-tts pygame SpeechRecognition pyaudio
```

👉 pyaudio hata verirse:

```bash
pip install pipwin
pipwin install pyaudio
```

---

### 3️⃣ Run

```bash
python hand_control.py
```

---

## 🎮 Controls

* ✋ Move → Mouse control
* 👆👆 → Left click
* ✊ → Right click
* 🖐️ → Scroll
* 👍👆 → Alt+Tab
* **V** → Voice command
* **Q** → Exit
